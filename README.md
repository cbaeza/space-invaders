# Space invader

Just playing and learning [pygame](pygame.org), I decide to create
my own game to demostrate in action.

# How to play

start the play with the command:

    $ python3 main.py 


# Keyboard shortcuts

    left: left arrow
    right: right arrow 
    up: up arrow
    down: down arrow
    fire: tab bar
